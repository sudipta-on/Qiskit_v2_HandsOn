# ⚛️ Qiskit v2 Hands-On

![Quantum](https://img.shields.io/badge/Qiskit-v2-blue)
![Python](https://img.shields.io/badge/Python-3.x-green)
![License](https://img.shields.io/badge/License-MIT-orange)

A quick and practical introduction to **Qiskit v2** for exploring the fundamentals of **Quantum Computing**.

This repository contains **hands-on notebooks and examples** demonstrating key quantum computing concepts using the modern **Qiskit SDK**.

---

## 🚀 What You Will Learn

Through these notebooks you will explore:

- Quantum circuits
- Superposition and entanglement
- Quantum parallelism
- Quantum algorithms
- Quantum cryptography (BB84)
- Simulation using **AerSimulator**
- Noise modeling in quantum circuits

---

| Notebook | Description |
|--------|-------------|

| **2. VQE Tutorial** | Exploring Variational Quantum EigenSolver |
